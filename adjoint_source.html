class pyadjoint.adjoint_source.AdjointSource(adj_src_type, misfit, dt, component, adjoint_source=None, network=None, station=None)[source]¶

Class representing an already calculated adjoint source.

Parameters:
  • adj_src_type (str) – The type of adjoint source.
  • misfit (float) – The misfit value.
  • dt (float) – The sampling rate of the adjoint source.
  • component (str) – The adjoint source component, usually "Z", "N", "E", "R", or "T".
  • adjoint_source (numpy.ndarray) – The actual adjoint source.
  • network (str) – The network code of the station.
  • station (str) – The station code of the station.
write(filename, format, **kwargs)[source]¶

Write the adjoint source to a file.

Parameters:
  • filename (str, open file, or file-like object) – Determines where the adjoint source is saved.
  • format (str) – The format of the adjoint source. Currently available are: "SPECFEM"

SPECFEM

SPECFEM requires one additional parameter: the temporal offset of the first sample in seconds. The following example sets the time of the first sample in the adjoint source to -10.

>>> adj_src.write("NET.STA.CHAN.adj", format="SPECFEM",
...               time_offset=-10)
pyadjoint.adjoint_source.calculate_adjoint_source(adj_src_type, observed, synthetic, min_period, max_period, left_window_border, right_window_border, adjoint_src=True, plot=False, plot_filename=None, **kwargs)[source]¶

Central function of Pyadjoint used to calculate adjoint sources and misfit.

This function uses the notion of observed and synthetic data to offer a nomenclature most users are familiar with. Please note that it is nonetheless independent of what the two data arrays actually represent.

The function tapers the data from left_window_border to right_window_border, both in seconds since the first sample in the data arrays.

Parameters:
  • adj_src_type (str) – The type of adjoint source to calculate.
  • observed (obspy.core.trace.Trace) – The observed data.
  • synthetic (obspy.core.trace.Trace) – The synthetic data.
  • min_period (float) – The minimum period of the spectral content of the data.
  • max_period (float) – The maximum period of the spectral content of the data.
  • left_window_border (float) – Left border of the window to be tapered in seconds since the first sample in the data arrays.
  • right_window_border (float) – Right border of the window to be tapered in seconds since the first sample in the data arrays.
  • adjoint_src (bool) – Only calculate the misfit or also derive the adjoint source.
  • plot (bool or empty matplotlib.figure.Figure instance) – Also produce a plot of the adjoint source. This will force the adjoint source to be calculated regardless of the value of adjoint_src.
  • plot_filename (str) – If given, the plot of the adjoint source will be saved there. Only used if plot is True.
